# vision-deck-proto

Node.js v16.15.1
yarn v1.22.18

## Project setup

```shell
npm install -g yarn
yarn install
```

### Compiles and hot-reloads for development

```shell
yarn run electron:serve
```

### Compiles and minifies for production

```shell
yarn run electron:build
```

### Lints and fixes files

```shell
yarn run lint
```
